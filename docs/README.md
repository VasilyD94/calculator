## Текущее состояние проекта

### Инфраструктура (все готово)

| Компонент | Статус | Детали |
|-----------|--------|--------|
| Домен | calc-box.ru | Куплен на REG.ru, DNS настроены |
| SSL | Let's Encrypt | Автообновление через certbot |

| Аналитика | Яндекс.Метрика | Счётчик 106583944, Вебвизор включён |
| Индексация | Яндекс.Вебмастер | Верификация пройдена, sitemap отправлен |
| Git | GitHub | Деплой одной командой |

### Калькуляторы — 13 штук на проде

| Раздел | Калькуляторы | Кол-во |
|--------|-------------|--------|
| **Питание** | Калории, БЖУ, Дефицит калорий, Норма воды | 4 |
| **Тело** | ИМТ, Идеальный вес, Процент жира, Базовый метаболизм | 4 |
| **Беременность** | Дата родов, Срок беременности, Овуляция | 3 |
| **Спорт** | Пульс для тренировок, Белок для спортсменов | 2 |

### Страницы — 20 на проде

1 главная + 4 хаба разделов + 13 калькуляторов + политика конфиденциальности + cookie-баннер = 20 страниц

Каждая страница калькулятора включает:
- Metadata (title, description, keywords, OG, canonical)
- JSON-LD (WebApplication, FAQPage, HowTo, BreadcrumbList)
- Хлебные крошки
- Калькулятор с реалтайм-расчётом
- SEO-текст (1500+ слов)
- FAQ
- Связанные калькуляторы
